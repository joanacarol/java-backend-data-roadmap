# Fase 1 — Fundamentos de Programação e Java (Base Absoluta)

## Conteúdo estudado

- Conceitos fundamentais de programação: algoritmo, automação e programas de computador  
- Estrutura de linguagens de programação e processo de execução (compilação, interpretação e máquina virtual)  
- Ecossistema Java: JDK, JVM e configuração do ambiente de desenvolvimento  
- Sintaxe básica da linguagem Java e estrutura de um programa  
- Estrutura sequencial: variáveis, tipos primitivos, operações, entrada e saída de dados  
- Estrutura condicional: `if/else`, operadores lógicos, `switch` e operador ternário  
- Estruturas repetitivas: `while`, `for` e `do-while`  
- Conceitos básicos adicionais: funções, manipulação inicial de `String`, convenções de código e operadores básicos

## Complementar aplicado

- Funcionamento da execução de programas na memória (**stack vs heap**)  
- Diferença entre **compilação, interpretação e execução na JVM**  
- Uso de **entrada e saída padrão (stdin / stdout)** no terminal
