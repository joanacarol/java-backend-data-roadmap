# Java Backend Roadmap

Repositório de estudos e projetos focado em **Backend Java com dados**,
incluindo **SQL, APIs REST e NoSQL**.

---

## 🎯 Objetivo

Construir base sólida em **Java Backend** com foco em:

- Modelagem de dados  
- Persistência  
- APIs REST  
- Boas práticas de desenvolvimento  

---

## 🚀 Tecnologias

- Java  
- JDBC  
- SQL (PostgreSQL)  
- Spring Boot  
- JPA / Hibernate  
- MongoDB  

---

## 📂 Organização

O repositório segue um **roadmap linear**, dividido por fases de aprendizado.

Cada pasta contém:

- 📘 README explicativo  
- 🧠 Exercícios autorais  
- 🛠 Projetos práticos
